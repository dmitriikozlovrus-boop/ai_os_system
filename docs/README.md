# Docs

Папка содержит проектную, техническую, сервисную и продуктовую документацию AI OS.

## Основные разделы

- `architecture/` — архитектура реализации и структура репозитория.
- `data/` — модели данных, схемы БД и источники истины.
- `services/` — документация рабочих сервисов.
- `product/` — продуктовые сценарии, требования и use cases.
- `roadmap/` — планы развития и функциональные карты.
- `decisions/` — принятые архитектурные решения.
- `general/` — общая документация.

## Отличие от knowledge

`docs/` содержит проектные и реализационные документы.

`knowledge/` содержит устойчивые знания, правила, сущности, классификацию и стандарты.
