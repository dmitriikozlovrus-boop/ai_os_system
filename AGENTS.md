# Правило хранения результатов AI Transformation

## Главное правило

Все новые документы, отчеты, планы, исследования, спецификации и другие
результаты работы по инициативе AI Transformation должны сохраняться локально
в:

`/Users/dmitrii/Documents/Codex/AI Transformation`

Материалы текущего проекта "ИИ Агент Дирижер" должны сохраняться в:

`/Users/dmitrii/Documents/Codex/AI Transformation/ИИ Агент Дирижер`

## Рабочий процесс

- Исходный код ведется в Git-репозитории проекта.
- Документы по проекту складываются в папку `Документы` внутри архива проекта.
- Для каждого нового проекта создается отдельная папка внутри `AI Transformation`.
- В ответе пользователю всегда указывается полный путь к созданному результату.
- Значимые изменения исходного кода должны быть закоммичены и отправлены на
  GitHub, когда пользователь просит опубликовать или сохранить их удаленно.
- Notion не используется для архива без отдельного прямого указания пользователя.
