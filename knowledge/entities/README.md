# Entities

Определения сущностей AI OS.

Сущности используются для классификации, хранения, маршрутизации и связывания информации внутри системы.

Каждый документ сущности отвечает на вопросы:

- Что представляет собой сущность.
- Зачем она существует.
- Когда должна создаваться.
- Когда не должна создаваться.
- Чем отличается от соседних сущностей.
- С какими сущностями обычно связана.

## Structure

Каждая сущность описывается отдельным документом.
